Basic Pin

You can pin an element while the trigger is active so that it stays in place between the defined start and end points.

Scroll down

start: "top top" triggers when the orange element’s top edge meets the viewport top. end: "bottom 150px" stops pinning when the orange element’s bottom is 150px from the top.

start: "top center" pins this element when the red element’s top reaches the viewport center, and remains pinned for 200px (end: "+=200").
